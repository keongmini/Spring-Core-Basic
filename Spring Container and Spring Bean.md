# 스프링 컨테이너, 스프링 빈

## 스프링 컨테이너
```ApplicationContext```
  - 스프링 컨테이너
  - 인터페이스 (다형성 적용)
  - 스프링 컨테이너는 XML기반, 애노테이션 기반의 자바 설정 클래스로 생성 가능하지만 대부분 애노테이션 기반을 사용

* 스프링 컨테이너 생성 과정
  1. ```new AnnotationConfigApplicationContext(AppConfig.class)```로 AppConfig 정보 전달
  2. 스프링 컨테이너 생성(구성 정보 전달 - AppConfig.class)
  3. 컨테이너의 스프링 빈 저장소에 '빈 이름'과 '빈 객체'를 key-value로 저장
  4. 스프링 빈 저장소에 스프링 빈 등록(@Bean)
     > 빈 이름: 기본적으로는 메소드 이름 사용(유니크 해야함)
  5. 스프링 빈 의존관계 설정 - 설정 정보를 참고해서 컨테이너가 의존관계를 주입해줌
 
  => 스프링 컨테이너 생성, 설정 정보 참고해서 스프링 빈 등록, 의존관계 설정 완료
